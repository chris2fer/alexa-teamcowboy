# alexa-teamcowboy
Alexa skill to integrate with Team Cowboy

##  Intent Ideas

- Get Next Game
- How many people playing
- Who is playing
- What was the score of the last game
- What is my status for the next game?
- How many more games do i need to play to be eligible for playoffs
- How many games have i played
- [When, Where] is the next game
- How many [goals, points, penalties, assists] do i have?
